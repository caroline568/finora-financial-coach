💰 Finora – AI-Powered Financial Coach for Kenyans

Helping Kenyans make smarter financial decisions through AI-powered budgeting, expense tracking, and personalized financial insights.

📖 Overview

Finora is a modern personal finance web application designed specifically for the Kenyan market. The platform empowers users to take control of their finances by tracking income and expenses, creating budgets, monitoring savings goals, and receiving AI-powered financial recommendations.

Built with a clean, responsive interface and modern web technologies, Finora demonstrates full-stack development skills, secure authentication, cloud database integration, and scalable application architecture. The project emphasizes usability, accessibility, and data-driven financial management.
